# OffWeGo ✈️

A travel blog website built by two engineering students from Tunisia, sharing destinations, travel stories, and tips for adventurers on any budget.

## 🌍 About

OffWeGo is a static travel blog featuring curated destinations across three categories — beach escapes, mountain & nature, and urban cities — alongside personal travel stories and a contact page.

## ✨ Features

- Scroll-reveal animations
- Destination filtering by category
- Photo lightbox gallery
- FAQ accordion
- Contact form with validation
- Smooth scrolling navigation
- Responsive mobile menu
- Logo animations

## 🛠️ Tech Stack

- HTML5 / CSS3
- Vanilla JavaScript (no frameworks)
- Bootstrap 5.3 (select pages)

## 🚀 Quick Start

No build tools required. Open `index.html` directly or serve locally:

| Page | Description |
